# 4.4 源码解析：pvPortMalloc / vPortFree trace

## heap_4 核心数据结构

### BlockLink_t（空闲块链表节点）

```c
typedef struct A_BLOCK_LINK {
    struct A_BLOCK_LINK *pxNextFreeBlock;  // 下一个空闲块
    size_t xBlockSize;                     // 块大小（含头部本身）
} BlockLink_t;
```

### 空闲链表初始化

```c
/* heap_4.c */
static BlockLink_t xStart;      // 链表头（始终存在，不代表实际堆）
static BlockLink_t *pxEnd = NULL;

void vPortInitializeHeap(void) {
    /* 初始化空闲链表头 */
    xStart.pxNextFreeBlock = (BlockLink_t *)&xStart;
    xStart.xBlockSize = (size_t)0;

    /* 计算堆起始地址 */
    xAlignedHeap = (uint8_t *)((((size_t)ucHeap) + portBYTE_ALIGNMENT - 1) & ~((size_t)portBYTE_ALIGNMENT_MASK));

    /* 计算堆结束地址（8字节对齐） */
    xAlignedHeap = (uint8_t *)((portPOINTER_SIZE_TYPE)xAlignedHeap & ~((portPOINTER_SIZE_TYPE)portBYTE_ALIGNMENT_MASK));
    pxEnd = (BlockLink_t *)((xAlignedHeap + configTOTAL_HEAP_SIZE) & ~((size_t)portBYTE_ALIGNMENT_MASK));

    /* 在末尾放置结束标记 */
    pxEnd->xBlockSize = 0;
    pxEnd->pxNextFreeBlock = (BlockLink_t *)xAlignedHeap;
}
```

### Step 4：插入空闲链表（按地址排序）

```c
static void prvInsertBlockIntoFreeList(BlockLink_t *pxBlockToInsert) {
    BlockLink_t *pxIterator;

    /* 从链表头开始找插入位置（按地址从低到高） */
    for (pxIterator = &xStart;
         pxIterator->pxNextFreeBlock < pxBlockToInsert;
         pxIterator = pxIterator->pxNextFreeBlock) {
        /* 空循环，找第一个 pxNextFreeBlock > pxBlockToInsert 的位置 */
    }

    /* 检查是否可以和前一块合并 */
    if (((BlockLink_t *)((uint8_t *)pxIterator) + pxIterator->xBlockSize) == pxBlockToInsert) {
        /* 前一块紧邻，合并 */
        pxIterator->xBlockSize += pxBlockToInsert->xBlockSize;
        pxBlockToInsert = pxIterator;
    }

    /* 检查是否可以和后一块合并 */
    if (((BlockLink_t *)((uint8_t *)pxBlockToInsert + pxBlockToInsert->xBlockSize)) == pxIterator->pxNextFreeBlock) {
        /* 后一块紧邻，合并 */
        pxBlockToInsert->xBlockSize += pxIterator->pxNextFreeBlock->xBlockSize;
        pxBlockToInsert->pxNextFreeBlock = pxIterator->pxNextFreeBlock->pxNextFreeBlock;
    } else {
        pxBlockToInsert->pxNextFreeBlock = pxIterator->pxNextFreeBlock;
    }

    /* 更新前一块的指针 */
    pxIterator->pxNextFreeBlock = pxBlockToInsert;
}
```

## vPortFree 源码 trace

### Step 1：无效指针检查

```c
void vPortFree(void *pv) {
    uint8_t *puc = (uint8_t *)pv;
    BlockLink_t *pxLink;

    if (pv != NULL) {
        /* 指针偏移到 BlockLink 头部 */
        puc -= heapSTRUCT_SIZE;
        pxLink = (BlockLink_t *)puc;

        /* 简单检查：块大小必须合理 */
        configASSERT(pxLink->xBlockSize > 0);
        configASSERT(pxLink->xBlockSize & (size_t)(portBYTE_ALIGNMENT_MASK));
    }
}
```

## 本节小结

- **BlockLink_t**：8 字节头部（下一个空闲块指针 + 块大小）
- **xStart**：链表头节点（空的头部，不代表实际堆）
- **pvPortMalloc**：遍历找第一个够大的块，可能分割剩余空间
- **vPortFree**：调用 `prvInsertBlockIntoFreeList`，自动合并相邻空闲块
- **合并时机**：释放时立即合并，防止碎片累积
