# 5.2 API 速查：信号量创建 / Take / Give

## 信号量创建

### xSemaphoreCreateBinary（二进制信号量）

```c
SemaphoreHandle_t xSemaphoreCreateBinary(void);
```

```c
// 创建后计数器为 0，需要先 Give 一次才能 Take
SemaphoreHandle_t xBinarySem = xSemaphoreCreateBinary();
// 此时：Take 会阻塞，Give 后 Take 才能成功
```

### xSemaphoreCreateMutex（互斥量）✅ 最常用

```c
SemaphoreHandle_t xSemaphoreCreateMutex(void);
```

```c
// 创建后计数器为 1（可用）
SemaphoreHandle_t xMutex = xSemaphoreCreateMutex();
```

### xSemaphoreCreateCounting（计数信号量）

```c
SemaphoreHandle_t xSemaphoreCreateCounting(
    UBaseType_t uxMaxCount,    // 最大计数值
    UBaseType_t uxInitialCount // 初始计数值
);
```

```c
// 最多 5 个资源，初始 3 个可用
SemaphoreHandle_t xCountSem = xSemaphoreCreateCounting(5, 3);
```

### xSemaphoreCreateRecursiveMutex（递归互斥量）

```c
SemaphoreHandle_t xSemaphoreCreateRecursiveMutex(void);
```

## Take 操作

### xSemaphoreTake（普通信号量/互斥量）

```c
BaseType_t xSemaphoreTake(
    SemaphoreHandle_t xSemaphore,  // 信号量句柄
    TickType_t xTicksToWait         // 超时时间
);
```

| 参数 | 效果 |
|------|------|
| `0` | 不等待，立即返回 |
| `portMAX_DELAY` | 永久等待（死等） |
| `pdMS_TO_TICKS(100)` | 等 100ms |

```c
// 成功：pdTRUE，失败：pdFALSE
if (xSemaphoreTake(xMutex, pdMS_TO_TICKS(100)) == pdTRUE) {
    // 成功获取，可以访问共享资源
    xSemaphoreGive(xMutex);  // 用完释放
} else {
    // 超时，没拿到
}
```

### xSemaphoreTakeRecursive（递归互斥量）

```c
BaseType_t xSemaphoreTakeRecursive(
    SemaphoreHandle_t xMutex,   // 递归互斥量句柄
    TickType_t xTicksToWait
);
```

```c
// 同一个任务可以多次获取递归互斥量
xSemaphoreTakeRecursive(xRecMutex, portMAX_DELAY);  // 第1次
xSemaphoreTakeRecursive(xRecMutex, portMAX_DELAY);  // 第2次（需要对应2次Give）
```

## Give 操作

### xSemaphoreGive（普通信号量/互斥量）

```c
BaseType_t xSemaphoreGive(SemaphoreHandle_t xSemaphore);
```

**注意**：
- 二进制信号量：Give 一次后 Take 才能成功
- 互斥量：Give 释放锁

### xSemaphoreGiveRecursive（递归互斥量）

```c
BaseType_t xSemaphoreGiveRecursive(SemaphoreHandle_t xMutex);
```

```c
// Give 次数必须等于 Take 次数
xSemaphoreTakeRecursive(xRecMutex, portMAX_DELAY);  // Take 1
xSemaphoreTakeRecursive(xRecMutex, portMAX_DELAY);  // Take 2
// ...
xSemaphoreGiveRecursive(xRecMutex);  // Give 1（还剩1次）
xSemaphoreGiveRecursive(xRecMutex);  // Give 2（全部释放）
```

### xSemaphoreGiveFromISR（中断中 Give）

```c
BaseType_t xSemaphoreGiveFromISR(
    SemaphoreHandle_t xSemaphore,
    BaseType_t *pxHigherPriorityTaskWoken
);
```

```c
// 中断中释放信号量
BaseType_t xHigherPriorityTaskWoken = pdFALSE;
xSemaphoreGiveFromISR(xBinarySem, &xHigherPriorityTaskWoken);

if (xHigherPriorityTaskWoken == pdTRUE) {
    portYIELD_FROM_ISR(xHigherPriorityTaskWoken);
}
```

## 查询操作

### uxSemaphoreGetCount（获取当前计数）

```c
UBaseType_t uxSemaphoreGetCount(SemaphoreHandle_t xSemaphore);
```

```c
UBaseType_t count = uxSemaphoreGetCount(xBinarySem);
// binary: 0 = 不可用, 1 = 可用
// counting: 返回当前计数
```

### xSemaphoreGetMutexHolder（获取互斥量持有者）

```c
TaskHandle_t xSemaphoreGetMutexHolder(SemaphoreHandle_t xMutex);
```

```c
TaskHandle_t holder = xSemaphoreGetMutexHolder(xMutex);
if (holder == xTaskGetCurrentTaskHandle()) {
    // 我持有这个互斥量
} else {
    // 其他任务持有
}
```

## 典型使用模式

### 1. 互斥保护共享资源

```c
SemaphoreHandle_t xLCDMutex;

void lcd_task(void *p) {
    xLCDMutex = xSemaphoreCreateMutex();

    while (1) {
        xSemaphoreTake(xLCDMutex, portMAX_DELAY);
        LCD_DrawSomething();
        xSemaphoreGive(xLCDMutex);
    }
}
```

### 2. 中断同步任务

```c
SemaphoreHandle_t xDataSem;

void IRQ_Handler(void) {
    BaseType_t xHigherPriorityTaskWoken = pdFALSE;
    xSemaphoreGiveFromISR(xDataSem, &xHigherPriorityTaskWoken);
    portYIELD_FROM_ISR(xHigherPriorityTaskWoken);
}

void data_task(void *p) {
    while (1) {
        xSemaphoreTake(xDataSem, portMAX_DELAY);
        process_data();
    }
}
```

### 3. 资源池

```c
SemaphoreHandle_t xUARTSem;

void init(void) {
    // 3 个缓冲区
    xUARTSem = xSemaphoreCreateCounting(3, 3);
}

void uart_send(uint8_t *buf) {
    // 获取一个缓冲区
    xSemaphoreTake(xUARTSem, portMAX_DELAY);

    // 发送数据
    HAL_UART_Transmit(&huart1, buf, strlen(buf), 100);

    // 释放缓冲区
    xSemaphoreGive(xUARTSem);
}
```

## 本节小结

- `xSemaphoreCreateMutex()` 是最常用的互斥量创建方式
- Take 用 `xSemaphoreTake`，Give 用 `xSemaphoreGive`
- 中断中用 `xSemaphoreGiveFromISR`，记得判断 `pxHigherPriorityTaskWoken`
- `portMAX_DELAY` = 永久等待，`pdMS_TO_TICKS(ms)` = 有限超时
- 递归互斥量可重复获取，但 Give 次数必须等于 Take 次数
