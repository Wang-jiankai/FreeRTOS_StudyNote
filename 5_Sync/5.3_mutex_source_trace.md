# 5.3 源码解析：xSemaphoreTake trace

> 本节源码 trace 基于 **FreeRTOS v11.1.0** queue.c 和 semphr.h

## 互斥量的本质

互斥量在 FreeRTOS 内部是**队列类型为 `queueQUEUE_IS_MUTEX` 的队列**（队列长度为 1，元素大小为 0）：

```c:semphr.h
#define xSemaphoreCreateMutex()    xQueueCreateMutex(queueQUEUE_TYPE_MUTEX)
```

### 互斥量初始化（queue.c）

```c:queue.c
QueueHandle_t xQueueCreateMutex(const uint8_t ucQueueType)
{
    Queue_t * pxNewQueue = xQueueGenericCreate((UBaseType_t)1,  /* 长度 1 */
                                               (UBaseType_t)0,  /* 元素大小 0 */
                                               ucQueueType);
    if (pxNewQueue != NULL) {
        pxNewQueue->u.xSemaphore.xMutexHolder = NULL;         /* 无人持有 */
        pxNewQueue->u.xSemaphore.uxRecursiveCallCount = 0;    /* 递归计数 0 */
        /* 将互斥量设为"已释放"状态（发送一次）*/
        (void) xQueueGenericSend(pxNewQueue, NULL, 0, queueSEND_TO_BACK);
    }
    return pxNewQueue;
}
```

```c:queue.c
BaseType_t xQueueSemaphoreTake(QueueHandle_t xQueue,
                                TickType_t xTicksToWait)
{
    BaseType_t xEntryTimeSet = pdFALSE;
    Queue_t * const pxQueue = xQueue;

    configASSERT(pxQueue->uxQueueType == queueQUEUE_IS_MUTEX);

    for (;;) {
        taskENTER_CRITICAL();

        /* 互斥量有持有者检查逻辑（与普通队列不同）*/
        if (pxQueue->uxMessagesWaiting == (UBaseType_t)0) {
            /* 互斥量未被持有（注意：互斥量空 = 已释放）*/
        }

        /* 阻塞等待（队列空则等待）*/
        if (/* 队列空 */) {
            vTaskPlaceOnUnorderedEventList(&(pxQueue->xTasksWaitingToReceive), xTicksToWait);

            vQueuePriorityInheritance(xQueue, xTicksToWait);
        }

        taskEXIT_CRITICAL();
        portYIELD();

        /* 超时检查... */
    }
}
```

## 优先级继承的实现

### 关键结构（queue.c）

```c:queue.c
/* 互斥量持有者记录在 SemaphoreData_t 中 */
typedef struct SemaphoreData {
    TaskHandle_t xMutexHolder;           /* 持有者的 TCB 句柄 */
    UBaseType_t uxRecursiveCallCount;   /* 递归嵌套次数 */
} SemaphoreData_t;

/* 互斥量通过 xQueue.uxMessagesWaiting 判断是否被持有
 * uxMessagesWaiting == 0 → 已释放（可用）
 * uxMessagesWaiting > 0  → 已被持有 */
```

### 优先级继承触发（vQueuePriorityInheritance）

```c:queue.c
void vQueuePriorityInheritance(QueueHandle_t xQueue,
                               TickType_t xTicksToWait)
{
    Task_t * pxHolder;
    Task_t * pxCurrentTask = xTaskGetCurrentTask();

    /* 如果互斥量已被持有 */
    if (pxQueue->u.xSemaphore.xMutexHolder != NULL) {
        pxHolder = pxQueue->u.xSemaphore.xMutexHolder;

        /* 持有者优先级 < 当前等待者优先级 → 提升持有者 */
        if (pxHolder->uxPriority < pxCurrentTask->uxPriority) {
            vTaskPrioritySet(pxHolder, pxCurrentTask->uxPriority);
        }
    }
}
```

### 互斥量释放时恢复优先级

```c:queue.c
/* 在 xQueueGenericSend 中，当互斥量被释放时 */
if (pxQueue->uxQueueType == queueQUEUE_IS_MUTEX) {
    /* 互斥量被 Give：从等待链表取下一个任务 */
    pxQueue->u.xSemaphore.xMutexHolder = pxWaitingTask;  /* 新持有者 */
    pxQueue->u.xSemaphore.uxRecursiveCallCount = 0;

}
```

## 递归互斥量的实现

### 持有者检查

```c
#define xSemaphoreTakeRecursive(xMutex, xBlockTime) \
    xQueueTakeMutexRecursive((xMutex), (xBlockTime))
```

### xQueueTakeMutexRecursive（queue.c）

```c:queue.c
BaseType_t xQueueTakeMutexRecursive(QueueHandle_t xQueue,
                                     TickType_t xTicksToWait)
{
    BaseType_t xReturn;

    /* 检查当前任务是否已经是持有者 */
    if (pxQueue->u.xSemaphore.xMutexHolder == xTaskGetCurrentTask()) {
        /* 已持有 → 递归获取，只累加计数 */
        pxQueue->u.xSemaphore.uxRecursiveCallCount++;
        xReturn = pdPASS;
    } else {
        /* 第一次获取，调用普通 Take 等待互斥量 */
        xReturn = xQueueSemaphoreTake(xQueue, xTicksToWait);

        if (xReturn == pdPASS) {
            /* 获取成功，计数初始化为 1 */
            pxQueue->u.xSemaphore.uxRecursiveCallCount = 1;
        }
    }

    return xReturn;
}
```

### xQueueGiveMutexRecursive（queue.c）

```c
#define xSemaphoreGiveRecursive(xMutex) \
    xQueueGiveMutexRecursive(xMutex)
```

```c:queue.c
BaseType_t xQueueGiveMutexRecursive(QueueHandle_t xQueue)
{
    /* 只有持有者才能释放 */
    if (pxQueue->u.xSemaphore.xMutexHolder == xTaskGetCurrentTask()) {
        pxQueue->u.xSemaphore.uxRecursiveCallCount--;

        /* 计数为 0 → 真正释放 */
        if (pxQueue->u.xSemaphore.uxRecursiveCallCount == 0) {
            (void) xQueueGenericSend(xQueue, NULL,
                                     queueMUTEX_GIVE_BLOCK_TIME,
                                     queueSEND_TO_BACK);
        }
        return pdPASS;
    }
    return pdFAIL;  /* 不是持有者，释放失败 */
}
```

## 本节小结

- **互斥量本质** = 队列类型标记为 `queueQUEUE_IS_MUTEX` 的队列，元素大小为 0
- **`xMutexHolder`**（`TaskHandle_t`）：记录当前持有者的 TCB 句柄
- **`uxRecursiveCallCount`**：递归嵌套计数，同一任务多次 Take 时累加
- **优先级继承**：`vQueuePriorityInheritance` 检测持有者优先级低于等待者时，提升持有者
- **递归 Take**：检查 `xMutexHolder == 当前任务`，相等则累加计数而非阻塞
- **递归 Give**：计数减到 0 时才真正释放（调用 `xQueueGenericSend`）
