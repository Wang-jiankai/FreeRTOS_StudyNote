# 9.2 API 速查：事件标志组

## 创建与删除

### xEventGroupCreate

```c:event_groups.h
EventGroupHandle_t xEventGroupCreate(void);
```

**返回值**：事件组句柄（失败返回 NULL）

```c
EventGroupHandle_t xEventGroup = xEventGroupCreate();
if (xEventGroup == NULL) {
    // 创建失败（堆内存不足）
}
```

```c:event_groups.h

```

## 置位与清除

### xEventGroupSetBits — 置位（发信号）

```c:event_groups.h
EventBits_t xEventGroupSetBits(EventGroupHandle_t xEventGroup,
                                const EventBits_t uxBitsToSet);
```

**参数**：`uxBitsToSet` = 要置位的掩码（如 `BIT_0 | BIT_1`）

**返回值**：置位后的事件组当前值

```c
// 置位 bit0
xEventGroupSetBits(xEventGroup, 0x01);

// 置位 bit0 和 bit2
xEventGroupSetBits(xEventGroup, 0x05);
```

### xEventGroupClearBits — 清除位

```c:event_groups.h
EventBits_t xEventGroupClearBits(EventGroupHandle_t xEventGroup,
                                   const EventBits_t uxBitsToClear);
```

## 等待

### xEventGroupWaitBits — 等待位满足条件

```c:event_groups.h
EventBits_t xEventGroupWaitBits(
    EventGroupHandle_t xEventGroup,
    const EventBits_t uxBitsToWaitFor,  /* 等待的位掩码 */
    const BaseType_t xClearOnExit,       /* pdTRUE = 退出时清除这些位 */
    const BaseType_t xWaitForAllBits,   /* pdTRUE = 等待所有位（AND）*/
    TickType_t xTicksToWait              /* 超时 */
);
```

**返回值**：等待条件满足时的当前事件值

```c
// 等待 bit0 或 bit1 任意一个（OR）
EventBits_t bits = xEventGroupWaitBits(
    xEventGroup,
    BIT_0 | BIT_1,      // 等待这些位
    pdFALSE,            // 不自动清除
    pdFALSE,            // OR（任意一个即可）
    portMAX_DELAY       // 无限等待
);

// 等待 bit0 AND bit1 全部就绪（AND）
EventBits_t bits = xEventGroupWaitBits(
    xEventGroup,
    BIT_0 | BIT_1,
    pdFALSE,
    pdTRUE,             // AND 模式
    pdMS_TO_TICKS(5000) // 5 秒超时
);
```

## 同步屏障（Barrier）

### xEventGroupSync — 同步所有任务

```c:event_groups.h
EventBits_t xEventGroupSync(EventGroupHandle_t xEventGroup,
                             const EventBits_t uxBitsToSet,
                             const EventBits_t uxBitsToWaitFor,
                             TickType_t xTicksToWait);
```

所有任务调用同一个 `xEventGroupSync`，等所有人都到齐才继续：

```c
// 所有任务都调用这个，才继续执行
xEventGroupSync(xEventGroup, BIT_MY_TASK, BITS_ALL_TASKS, portMAX_DELAY);
// 到达这里说明所有任务都同步了
```

## 完整使用示例

```c
#define BIT_SENSOR_TEMP  (1UL << 0)
#define BIT_SENSOR_HUMI  (1UL << 1)
#define BIT_BUTTON       (1UL << 2)

EventGroupHandle_t xEventGroup;

void sensor_temp_task(void *p) {
    while (1) {
        read_temp();
        xEventGroupSetBits(xEventGroup, BIT_SENSOR_TEMP);
        vTaskDelay(pdMS_TO_TICKS(1000));
    }
}

void sensor_humi_task(void *p) {
    while (1) {
        read_humidity();
        xEventGroupSetBits(xEventGroup, BIT_SENSOR_HUMI);
        vTaskDelay(pdMS_TO_TICKS(2000));
    }
}

void process_task(void *p) {
    while (1) {
        // 等待温度和湿度都就绪
        xEventGroupWaitBits(xEventGroup,
                            BIT_SENSOR_TEMP | BIT_SENSOR_HUMI,
                            pdTRUE,   // 清除已等待的位
                            pdTRUE,   // AND：两个都要
                            portMAX_DELAY);
        // 两个传感器数据都准备好了，执行处理
        process_data();
    }
}
```

## 本节小结

| 函数 | 作用 |
|------|------|
| `xEventGroupCreate()` | 创建事件组 |
| `xEventGroupSetBits()` | 置位（通知事件发生） |
| `xEventGroupClearBits()` | 清除位 |
| `xEventGroupWaitBits()` | 等待位满足条件（OR/AND） |
| `xEventGroupSync()` | 同步多个任务 |
| `vEventGroupDelete()` | 删除事件组 |
