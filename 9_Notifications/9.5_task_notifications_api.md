# 9.5 API 速查：任务通知

## 通知发送端

### xTaskNotifyGive — 递增通知值（替代信号量 Give）

```c:task.h
/* 递增目标任务的通知值（相当于 Give 一个信号量）*/
#define xTaskNotifyGive(xTaskToNotify) \
    xTaskGenericNotify((xTaskToNotify), (tskDEFAULT_INDEX_TO_NOTIFY), \
                        (0), eIncrement, NULL)
```

**效果**：目标任务的通知值 +1，任务从等待中唤醒。

### xTaskNotify — 通用通知

```c:task.h
BaseType_t xTaskNotify(TaskHandle_t xTaskToNotify,
                        uint32_t ulValue,
                        eNotifyAction eAction);
```

**eNotifyAction**（通知动作）：

| 动作 | 说明 |
|------|------|
| `eNoAction` | 不改变值，只唤醒任务（最省资源） |
| `eSetBits` | `ulValue` 按位 OR 到通知值 |
| `eIncrement` | 通知值 +1（等价于 `xTaskNotifyGive`）|
| `eSetValueWithOverwrite` | 直接覆盖（即使上次未读）|
| `eSetValueWithoutOverwrite` | 覆盖（只有未读时才写）|

```c
// 例 1：只唤醒，不传值（最快）
xTaskNotify(xTask, 0, eNoAction);

// 例 2：传数据（目标任务 ulTaskNotifyTake 得到的就是这个值）
xTaskNotify(xTask, sensor_value, eSetValueWithOverwrite);

// 例 3：按位或
xTaskNotify(xTask, BIT_0 | BIT_1, eSetBits);
```

### xTaskNotifyFromISR

```c:task.h
BaseType_t xTaskNotifyFromISR(TaskHandle_t xTaskToNotify,
                                uint32_t ulValue,
                                eNotifyAction eAction,
                                BaseType_t *pxHigherPriorityTaskWoken);
```

```c
void UART_IRQHandler(void) {
    BaseType_t xHigherPriorityTaskWoken = pdFALSE;
    uint8_t ch = USART1->DR;

    /* 直接通知处理任务，传递接收到的字节 */
    xTaskNotifyFromISR(xTaskToNotify, ch, eSetValueWithOverwrite,
                       &xHigherPriorityTaskWoken);

    portYIELD_FROM_ISR(xHigherPriorityTaskWoken);
}
```

## 通知接收端

### ulTaskNotifyTake — 等待通知（替代信号量 Take）

```c:task.h
uint32_t ulTaskNotifyTake(BaseType_t xClearCountOnExit,
                            TickType_t xTicksToWait);
```

**参数**：
- `xClearCountOnExit = pdTRUE`：唤醒后清除计数（清零）
- `xClearCountOnExit = pdFALSE`：只减 1

**返回值**：当前通知值（唤醒时）

```c
void task(void *p) {
    while (1) {
        /* 等待通知（类似 xSemaphoreTake）*/
        uint32_t notify_value = ulTaskNotifyTake(
            pdTRUE,            /* 唤醒后清零 */
            portMAX_DELAY      /* 无限等待 */
        );

        /* notify_value 就是发送端传来的值 */
        if (notify_value > 0) {
            process_data();
        }
    }
}
```

### xTaskNotifyWait — 等待任意通知（更灵活）

```c:task.h
BaseType_t xTaskNotifyWait(uint32_t ulBitsToClearOnEntry,
                             uint32_t ulBitsToClearOnExit,
                             uint32_t *pulNotificationValue,
                             TickType_t xTicksToWait);
```

**参数**：
- `ulBitsToClearOnEntry`：进入时清除哪些位
- `ulBitsToClearOnExit`：退出时清除哪些位

```c
void task(void *p) {
    uint32_t notify_value;

    while (1) {
        BaseType_t result = xTaskNotifyWait(
            0,                    /* 进入时不清除任何位 */
            BIT_0 | BIT_1,       /* 退出时清除 BIT_0 和 BIT_1 */
            &notify_value,        /* 输出：收到的通知值 */
            portMAX_DELAY
        );

        if (result == pdTRUE) {
            if (notify_value & BIT_0) process_temp();
            if (notify_value & BIT_1) process_humidity();
        }
    }
}
```

## 高级：多通知通道（v10.4+）

FreeRTOS v10.4 起，每个任务支持**多个独立的通知通道**：

```c:task.h
#define configTASK_NOTIFICATION_ARRAY_ENTRIES  4  /* 默认 4 个通道 */

/* 按索引通知 */
xTaskNotifyIndexed(xTask, 1, ulValue, eSetValueWithOverwrite);
xTaskNotifyIndexedFromISR(xTask, 1, ulValue, eAction, &xHigherPriorityTaskWoken);

/* 按索引等待 */
ulTaskNotifyTakeIndexed(1, pdTRUE, portMAX_DELAY);
xTaskNotifyWaitIndexed(1, ulBitsToClearOnEntry, ulBitsToClearOnExit,
                       &pulNotificationValue, xTicksToWait);
```

## 完整使用示例

### 场景：UART 接收处理

```c
TaskHandle_t xUartTaskHandle;

void uart_task(void *p) {
    while (1) {
        /* 等待通知（收到数据时 ISR 会 notify）*/
        uint32_t ch = ulTaskNotifyTake(pdTRUE, portMAX_DELAY);

        /* 处理接收到的字节 */
        rx_buffer[rx_head++] = (uint8_t)ch;
        if (rx_head >= 256) rx_head = 0;
    }
}

void USART1_IRQHandler(void) {
    BaseType_t xHigherPriorityTaskWoken = pdFALSE;

    if (USART1->SR & USART_SR_RXNE) {
        uint8_t ch = USART1->DR;

        /* 直接通知 UART 任务，不需要信号量 */
        xTaskNotifyFromISR(xUartTaskHandle, ch,
                           eSetValueWithOverwrite,
                           &xHigherPriorityTaskWoken);
    }

    portYIELD_FROM_ISR(xHigherPriorityTaskWoken);
}
```

| `xTaskNotifyGive()` | 发送 | ❌ | 递增 |

| `xTaskNotifyGiveIndexed()` | 发送 | ❌ | 递增（指定通道）|

| `ulTaskNotifyTake()` | 接收 | ❌ | 等待，值减 1 |

| `xTaskNotifyWaitIndexed()` | 接收 | ❌ | 等待（指定通道）|
