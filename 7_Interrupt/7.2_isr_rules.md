## 优先级设置

```c
// 设置中断优先级（数值越小优先级越高）
NVIC_SetPriority(UART1_IRQn, 5);  // 优先级 5
NVIC_SetPriority(TIM2_IRQn, 3);   // 优先级 3（更高）
```

## FreeRTOS 中的中断规则

### 规则 1：FromISR API 只能用于低于 configMAX_SYSCALL_INTERRUPT_PRIORITY 的中断

```c
#define configMAX_SYSCALL_INTERRUPT_PRIORITY  5
```

### 规则 2：FromISR 返回值判断

```c
BaseType_t xHigherPriorityTaskWoken = pdFALSE;
xQueueSendFromISR(xQueue, &data, &xHigherPriorityTaskWoken);

if (xHigherPriorityTaskWoken == pdTRUE) {
    portYIELD_FROM_ISR();  // 必须判断并触发切换
}
```

## 常见错误

### 错误 1：中断优先级设错

```c
// ❌ 优先级 3 < 5，不能调 FromISR
NVIC_SetPriority(UART1_IRQn, 3);
void UART_IRQHandler(void) {
    xQueueSendFromISR(xQueue, &data, NULL);  // ❌ 危险！
}

// ✅ 正确：优先级要 > 5
NVIC_SetPriority(UART1_IRQn, 6);
```

### 错误 2：忘记 portYIELD_FROM_ISR

```c
// ❌ 危险：高优先级任务可能无法及时唤醒
void UART_IRQHandler(void) {
    xQueueSendFromISR(xQueue, &data, NULL);  // 没判断返回值
}

// ✅ 正确
void UART_IRQHandler(void) {
    BaseType_t xHigherPriorityTaskWoken = pdFALSE;
    xQueueSendFromISR(xQueue, &data, &xHigherPriorityTaskWoken);

    if (xHigherPriorityTaskWoken == pdTRUE) {
        portYIELD_FROM_ISR();
    }
}
```

---

## 本节小结

- Cortex-M 优先级数值越小越高
- `FromISR` API 只能用于优先级 > `configMAX_SYSCALL_INTERRUPT_PRIORITY` 的中断
- 每次 FromISR 调用后必须判断 `pxHigherPriorityTaskWoken` 并调用 `portYIELD_FROM_ISR()`
