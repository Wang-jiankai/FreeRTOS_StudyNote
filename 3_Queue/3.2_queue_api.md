# 3.2 API 速查：队列创建 / 发送 / 接收

## 创建与删除

### xQueueCreate

```c
QueueHandle_t xQueueCreate(
    UBaseType_t uxQueueLength,  // 队列深度（能存多少个元素）
    UBaseType_t uxItemSize       // 每个元素的大小（字节）
);
```

**返回值**：`QueueHandle_t`（成功）或 `NULL`（失败，内存不足）

```c
// 示例：创建能存 10 个 uint32_t 的队列
QueueHandle_t xQueue = xQueueCreate(10, sizeof(uint32_t));
if (xQueue == NULL) {
    // 创建失败
}

// 示例：创建能存 5 个结构体的队列
typedef struct {
    uint8_t id;
    float value;
} SensorData;

QueueHandle_t xSensorQueue = xQueueCreate(5, sizeof(SensorData));
```

### xQueueCreateStatic（静态创建）⚠️ 少用

```c
QueueHandle_t xQueueCreateStatic(
    UBaseType_t uxQueueLength,
    UBaseType_t uxItemSize,
    uint8_t *pucQueueStorageBuffer,  // 用户分配的队列存储区
    StaticQueue_t *pxQueueBuffer     // 用户分配的队列控制块
);
```

### vQueueDelete

```c
void vQueueDelete(QueueHandle_t xQueue);
```

**注意**：删除队列不会唤醒等待中的任务，可能造成内存泄漏或任务永久阻塞。

## 发送数据

### xQueueSend（发送到尾部）✅ 最常用

```c
BaseType_t xQueueSend(
    QueueHandle_t xQueue,           // 队列句柄
    const void *pvItemToQueue,     // 数据指针（const，发送后数据不变）
    TickType_t xTicksToWait         // 超时时间
);
```

**返回值**：`pdTRUE`（成功）或 `errQUEUE_FULL`（超时）

```c
// 发送，队列满则等 100ms
uint32_t data = 42;
if (xQueueSend(xQueue, &data, pdMS_TO_TICKS(100)) == pdTRUE) {
    // 发送成功
}
```

### xQueueSendToBack（发送到尾部，同 xQueueSend）

```c
// xQueueSendToBack 和 xQueueSend 完全等价
xQueueSendToBack(xQueue, &data, portMAX_DELAY);
```

### xQueueSendToFront（发送到头部）

```c
// 数据插入队列头部，会被下一次接收
xQueueSendToFront(xQueue, &data, portMAX_DELAY);
```

### xQueueSendFromISR（中断中发送）

```c
BaseType_t xQueueSendFromISR(
    QueueHandle_t xQueue,
    const void *pvItemToQueue,
    BaseType_t *pxHigherPriorityTaskWoken  // 输出：是否需要任务切换
);
```

```c
void UART_IRQHandler(void) {
    BaseType_t xHigherPriorityTaskWoken = pdFALSE;
    uint8_t ch = USART1->DR;

    xQueueSendFromISR(xQueue, &ch, &xHigherPriorityTaskWoken);

    if (xHigherPriorityTaskWoken == pdTRUE) {
        portYIELD_FROM_ISR(xHigherPriorityTaskWoken);
    }
}
```

## 接收数据

### xQueueReceive ✅ 最常用

```c
BaseType_t xQueueReceive(
    QueueHandle_t xQueue,         // 队列句柄
    void *pvBuffer,               // 接收缓冲区（出参）
    TickType_t xTicksToWait       // 超时时间
);
```

**返回值**：`pdTRUE`（成功收到）或 `errQUEUE_EMPTY`（超时）

```c
// 接收，队列空则等 100ms
uint32_t received;
if (xQueueReceive(xQueue, &received, pdMS_TO_TICKS(100)) == pdTRUE) {
    // received 中是收到的数据
} else {
    // 超时，没收到
}
```

### xQueueReceiveFromISR（中断中接收）

```c
BaseType_t xQueueReceiveFromISR(
    QueueHandle_t xQueue,
    void *pvBuffer,
    BaseType_t *pxHigherPriorityTaskWoken
);
```

**注意**：FromISR 版本**不能阻塞**，队列空则立即返回 `errQUEUE_EMPTY`。

## 查询

### uxQueueMessagesWaiting

```c
UBaseType_t uxQueueMessagesWaiting(QueueHandle_t xQueue);
```

**返回**：队列中当前消息数量

```c
UBaseType_t count = uxQueueMessagesWaiting(xQueue);
if (count > 0) {
    // 队列有数据
}
```

### uxQueueSpacesAvailable

```c
UBaseType_t uxQueueSpacesAvailable(QueueHandle_t xQueue);
```

**返回**：队列剩余可存空间

## 典型使用模式

### 生产者-消费者

```c
// 生产者任务（发送传感器数据）
void sensor_task(void *param) {
    SensorData data;
    while (1) {
        data = read_sensor();
        xQueueSend(xQueue, &data, portMAX_DELAY);
    }
}

// 消费者任务（处理数据）
void process_task(void *param) {
    SensorData data;
    while (1) {
        if (xQueueReceive(xQueue, &data, portMAX_DELAY) == pdTRUE) {
            process_sensor_data(data);
        }
    }
}
```

### 中断触发处理

```c
// 中断产生数据
void UART_IRQHandler(void) {
    static uint8_t rx_buf[256];
    static uint8_t rx_head = 0;

    rx_buf[rx_head++] = USART1->DR;
    if (rx_head >= 256) rx_head = 0;

    BaseType_t xHigherPriorityTaskWoken = pdFALSE;
    xQueueSendFromISR(xQueue, &rx_buf, &xHigherPriorityTaskWoken);

    portYIELD_FROM_ISR(xHigherPriorityTaskWoken);
}
```

## 本节小结

- `xQueueCreate(深度, 元素大小)` 创建队列，返回 `QueueHandle_t`
- 发送用 `xQueueSend`，接收用 `xQueueReceive`，都带超时参数
- `xQueueSendToFront` 可以插队到头部（紧急数据）
- 中断中用 `FromISR` 版本，返回值指示是否需要 `portYIELD_FROM_ISR`
- `uxQueueMessagesWaiting()` 可以查询队列中现有消息数量
